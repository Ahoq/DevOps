Implementation Background

The service which we choose to use will highly depend on requirement of development and deployment team.

Cloud services come in three forms- SAAS,PAAS, IAAS.

SAAS- When software developed by cloud service provider is used to manage technical task of an organization.

PAAS- when cloud service provider provides an environment to host client's software.

IAAS - when whole computing infrastructure is provided by cloud service provider.

We decided to go with following cloud service providers-


SAAS - Salesforce
PAAS- Heroku
IAAS - AWS
