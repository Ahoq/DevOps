Our Evaluation of Jenkins

Jenkins is an open source automation tool written in Java which supports continuous integration.Hudson project was the precursor to Jenkins which was orignally started by Sun Systems, after the acquisition of Sun Systems by Oracle, Jenkins was forked from Hudson and it's independent community was formed. Jenkins is widely used by various software industries and the software folks love it.

Why Jenkins is desirable?

  • Jenkins is an automation tool that spans across the entire development cycle from Building to Deployment.
  • Jenkins has over 1400 plugins that makes is easily accessible across different platforms and frameworks.
  • It has a master-slave architechture and allows us to create nodes depending on the task in hand i.e you can create different jenkins node to handle tasks such as build,delpoy,test etc.
  • Jenkins supports git and many of the version controls for Source code management.
  • It is most active automation server tool installed in the software industry and has an estimation of more than 1 million active users using Jenkins.
  • Jenkins uses less memory for installation.
  • Jenkins can be installed on almost all of the operating systems.
  • The Jenkins Community is vastly large and hence we can get support, if for any issues
